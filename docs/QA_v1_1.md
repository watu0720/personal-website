# v1.1 受入チェックリスト（QA）

v1.1 修正反映後の最低限の確認項目です。

---

## 認証・表示

- [ ] ログイン後、ヘッダーに **Google アイコン** が表示される
- [ ] ヘッダーの **メールアドレスが小さめ**（text-xs 程度）で表示される
- [ ] ログインユーザーが投稿したコメントに **Google アイコン・表示名** が反映されている

---

## コメント・ハート

- [ ] **管理者**でログイン時、通常 HP のコメント一覧で **ハートの ON/OFF** ができる
- [ ] 非管理者・ゲストではハート操作ができない（または表示されない）
- [ ] **ログインユーザー**は自分のコメントに **編集ボタン** が表示され、編集できる
- [ ] **ゲスト**は投稿直後に **edit_token** を保持した端末でのみ編集でき、編集すると「編集済み」が分かる（任意表示）

---

## Home

- [ ] **最新の通知**が **縦 1 列** で表示されている
- [ ] **改訂履歴**が最新 5 件表示され、**もっと見る**でモーダルから全件閲覧できる
- [ ] YouTube / ニコニコ / GitHub の **外部マイページへのリンクボタン** が配置されている

---

## YouTube・ニコニコ

- [ ] 投稿動画は **最新 6 件** 表示
- [ ] **もっと見る** で **HP 内の各サービスの投稿一覧ページ**（`/youtube/videos`, `/niconico/videos`）へ遷移する
- [ ] ニコニコのマイリストは **各リスト最新 5 件** ＋ **マイリストページへ** のリンクボタンがある

---

## UI・文言

- [ ] ヘッダーのタブが **日本語**（ホーム / プロフィール / YouTube / ニコニコ / 開発）になっている
- [ ] 管理者画面の主要ラベル・ボタン・選択肢が **日本語** になっている
- [ ] 各ページの H1 の **左にアイコン** が表示されている（任意）

---

## プロフィール・レスポンシブ

- [ ] プロフィールカード上部に **グラデーションヘッダー**（赤→オレンジ→黄）がある
- [ ] **768px 未満**で文字が 1 段階小さくなり、主要レイアウトが崩れない

---

## 管理者・ストレージ

- [ ] 管理者画面に **ストレージ使用量の警告バナー** が表示される（しきい値 80% / 90% 等）
- [ ] **ストレージ整理**で「不要画像（orphan）」のプレビュー・削除ができる
- [ ] **ストレージ整理**で **audit_logs** の古いログを期間指定で削除できる
- [ ] **改訂履歴（Changelog）** が管理者画面から CRUD でき、Home に反映される

## セキュリティ・運用
